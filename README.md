# My AI Conversations Backup

## Project Overview

**My AI Conversations Backup** is a project designed to help users securely archive their conversations with artificial intelligence systems. This repository maintains a comprehensive backup of all interactions, ensuring that users have a reference to their discussions, insights, and the evolution of their dialogue with AI. 

### Features
- **Secure Storage**: Conversations are stored in a secure manner, keeping user data protected.
- **Easy Retrieval**: Users can easily access and review their past conversations without hassle.
- **User-friendly Interface**: The project provides a simple interface for managing conversations.
- **Version Control**: Changes and updates to conversations are tracked, allowing users to see how discussions have progressed over time.

### Technologies Used
- **Backend**: Python Flask for server-side logic.
- **Frontend**: React for a responsive user interface.
- **Database**: PostgreSQL for secure storage of conversation data.
- **Deployment**: Docker for containerized application deployment.

### Installation
To get started, clone the repository and install the required packages:
```bash
git clone https://github.com/Paym4It/my-ai-conversations-backup.git\
cd my-ai-conversations-backup\
pip install -r requirements.txt
```

### Usage
Run the application locally using:
```bash
flask run
```
Visit `http://localhost:5000` in your browser to start using the application.
